Filemanager by simogeo

An open-source file manager - started by Core Five Labs and few others contributors. This version is localized and includes many bug fixes and improvements.

FileManager is an open-source alternative to ckFinder. It does work correctly and can connect easily with ckEditor. It alson can be use as standalone version.
